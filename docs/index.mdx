# Yeoubi UI

Highly Customizable React UI Kit for Yeoubi

# Getting Started

## Installation

```console
yarn add @yeoubi/ui
```

## Usage

```typescript
import React from 'react';
import { Heading } from '@yeoubi/ui';

const Logo: React.FunctionCompoment<{}> = () => (
  <Heading level={1}>
    Yeoubi
  </Heading>
);
```
