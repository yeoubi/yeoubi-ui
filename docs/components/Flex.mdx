# Flex

다른 콘텐츠를 감싸는 `<Flex />` 컴포넌트입니다. `<Box />` 컴포넌트와 비슷하지만 세세한 정렬 기능이 있습니다.

<Props of={Flex} />

## Usage

<Playground>
  <Flex>
    <Body level={2}>
      Lorem Ipsum
    </Body>
  </Flex>
</Playground>

## inline

<Playground>
  <Space p={8} mr={8} mb={8}>
    <Flex bgColor="#000">
      <Body color="#FFF" level={2}>
        Lorem Ipsum
      </Body>
    </Flex>
  </Space>
  <Space p={8}>
    <Flex bgColor="#000" inline>
      <Body color="#FFF" level={2}>
        Lorem Ipsum
      </Body>
    </Flex>
  </Space>
</Playground>


## Direction

콘텐츠를 가로 또는 세로로 정렬할 수 있습니다.

<Playground>
  <Space p={8} mb={8}>
    <Flex bgColor="#F1F1F1">
      <Body level={2}>
        Lorem Ipsum
      </Body>
    </Flex>
  </Space>
  <Space p={8}>
    <Flex bgColor="#F1F1F1" vertical>
      <Body level={2}>
        Lorem Ipsum
      </Body>
    </Flex>
  </Space>
</Playground>
