# Flex

다른 콘텐츠를 감싸는 `<Flex />` 컴포넌트입니다. `<Box />` 컴포넌트와 비슷하지만 `display: flex` 스타일이 적용되어 있습니다.

<Props of={Flex} />

## Usage

<Playground>
  <Flex
    p={16}
    mb={16}
    bgColor="#F1F1F1"
    border="#E0E0E0"
  >
    <Box p={8} mr={8} bgColor="#FFFFFF">
      Lorem Ipsum
    </Box>
    <Box p={8} bgColor="#FFFFFF">
      dolor sit amet
    </Box>
  </Flex>
  <Flex
    p={16}
    display="inline-flex"
    bgColor="#F1F1F1"
    border="#E0E0E0"
  >
    <Box p={8} mr={8} bgColor="#FFFFFF">
      Lorem Ipsum
    </Box>
    <Box p={8} bgColor="#FFFFFF">
      dolor sit amet
    </Box>
  </Flex>
</Playground>
