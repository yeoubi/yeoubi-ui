# Heading

제목으로 사용할 수 있는 컴포넌트입니다.

<Props of={Heading} />

## Usage

<Playground>
  <Heading level={1} mb={8}>
    Lorem Ipsum
  </Heading>
  <Heading level={2} mb={8}>
    Lorem Ipsum
  </Heading>
  <Heading level={3} mb={8}>
    Lorem Ipsum
  </Heading>
  <Heading level={4} mb={8}>
    Lorem Ipsum
  </Heading>
  <Heading level={5} mb={8}>
    Lorem Ipsum
  </Heading>
</Playground>

## With other element

`as` 속성을 통해 element를 지정할 수 있습니다.

<Playground>
  <Heading level={2} as="h1">
    Lorem Ipsum
  </Heading>
</Playground>
