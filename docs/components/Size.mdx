# Size

별도의 스타일시트를 생성하지 않고 컴포넌트 크기를 조정합니다. **하나의 자식 컴포넌트**만 가질 수 있습니다.

<Props of={Size} />

## Usage

<Playground>
  <Space mb={16}>
    <Size width={100} height={50}>
      <div style={{ backgroundColor: '#000' }} />
    </Size>
  </Space>
  <Size width={[100, 200, 300]} height={50}>
    <div style={{ backgroundColor: '#000' }} />
  </Size>
</Playground>
