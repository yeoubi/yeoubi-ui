# Button

<Props of={Button} />

## Usage

<Playground>
  <Space pv={8} ph={16} mr={8}>

  <Space pv={8} ph={16} mr={8}>

## Use as anchor

`to`, `href` 속성을 넣어 `<a>` 태그로 사용할 수 있습니다. 앱 내 페이지에는 `to`, 외부 페이지에는 `href`를 사용하세요.

<Playground>
  <Space pv={8} ph={16}>
    <Button href="/components/button#usage" level={1} color="#FFF" bgColor="#1A1A1A">
      Lorem Ipsum
    </Button>
  </Space>
</Playground>
