import {
  Playground,
  Props,
} from 'docz';
import {
  Box,
  Body,
  Space,
} from '@yeoubi/ui';

# Box

다른 콘텐츠를 감싸는 박스 컴포넌트입니다.

<Props of={Box} />

## Usage

<Playground>
  <Box>
    <Body level={2}>
      Lorem Ipsum
    </Body>
  </Box>
</Playground>

## Common

`flex`, `overflow`, `inline` 속성을 설정합니다.

<Playground>
  <Space p={8} mr={8} mb={8}>
    <Box bgColor="#000">
      <Body color="#FFF" level={2}>
        Lorem Ipsum
      </Body>
    </Box>
  </Space>
  <Space p={8}>
    <Box bgColor="#000" inline>
      <Body color="#FFF" level={2}>
        Lorem Ipsum
      </Body>
    </Box>
  </Space>
</Playground>


## Border

테두리 색, 두께, 지름을 설정합니다.

<Playground>
  <Space p={8} mr={8}>
    <Box border="#000" borderWidth={2} inline>
      <Body level={2}>
        Lorem Ipsum
      </Body>
    </Box>
  </Space>
  <Space p={8} mr={8}>
    <Box border="#000" borderWidth={2} borderRadius={8} inline>
      <Body level={2}>
        Lorem Ipsum
      </Body>
    </Box>
  </Space>
  <Space p={8}>
    <Box border="0.125rem solid #000" inline>
      <Body level={2}>
        Lorem Ipsum
      </Body>
    </Box>
  </Space>
</Playground>

## Background

배경 색, 이미지를 설정합니다.

<Playground>
  <Space p={8} mr={8}>
    <Box bgColor="#000" inline>
      <Body color="#FFF" level={2}>
        Lorem Ipsum
      </Body>
    </Box>
  </Space>
</Playground>
